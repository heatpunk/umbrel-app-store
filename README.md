# Heatpunk Community App Store for Umbrel

Apps from [heatpunk](https://github.com/heatpunk), installable on any Umbrel.

## Install

On your Umbrel: **App Store → ⋯ → Community App Stores**, paste:

```
https://github.com/heatpunk/umbrel-app-store
```

Open the **Heatpunk** store and install **Hashboard**.

## Apps

| App | Description |
| --- | --- |
| [Hashboard](https://github.com/heatpunk/hashboard) | Run your ASIC miner as a space heater |

The app definitions here mirror `umbrel/` in the main
[hashboard](https://github.com/heatpunk/hashboard) repo and are updated on
each release.
